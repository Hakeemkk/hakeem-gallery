## 🚀 كيفية الاستخدام

### ابدأ التطبيق:
```bash
docker-compose up --build
```

### اختبر الاتصال:
```bash
# الفرونت إند
curl http://localhost:3030

# الباك إند
curl http://localhost:5000/api/products

# الـ Health Check
curl http://localhost:5000/health
```

### تحقق من السجلات:
```bash
docker-compose logs -f backend
```
